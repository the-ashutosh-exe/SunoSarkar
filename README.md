<br />
While ordinary apps merely transcribe photos to text, SunoSarkar AI solves human problems at scale. This flagship system operates as a **Hyper-Local Civic Action Engine**, bridging the gap between municipal authorities and citizens.

By leveraging Gemini AI Vision Grounding, 50m Spatial Clustering, and WMO Storm Radar integration, it automatically triages civic reports, evaluates severity, and dispatches them efficiently—effectively eliminating triage bottlenecks.
</details>

### ✨ Key Features

- **🛡️ VIP Dispatch Dashboard:** A dedicated command center for municipal judges to review AI-clustered reports.
- **⚡ Zero Triage Bottlenecks:** AI-driven incident grouping using a powerful 50m Spatial Clustering algorithm to prevent duplicate reports.
- **🤖 Gemini AI Vision Grounding:** Advanced computer vision automatically validates images attached to citizen complaints, extracting context, severity, and urgency.
- **🎯 Gamified Civic Engagement:** Users earn XP, level up, and gain reputation badges for contributing valid civic reports.
- **🌗 Adaptive UI Themes:** Ships with 'GenZ Dark Mode' and 'Solar Light' modes for high visibility under direct sunlight during outdoor civic surveys.

### ⚙️ Getting Started

#### Prerequisites
- Node.js `v18.x` or higher
- A Firebase Project (Auth, Firestore, Storage)
- Google Gemini API Key

#### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/the-ashutosh-exe/SunoSarkar.git
   cd SunoSarkar
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Configure Environment:**
   Copy the `.env.example` file and fill in your actual API credentials:
   ```bash
   cp .env.example .env.local
   ```
   > **Note:** Never commit your `.env.local` to version control.

4. **Start the Development Server:**
   ```bash
   npm run dev
   ```

5. **Build for Production:**
   ```bash
   npm run build
   ```
